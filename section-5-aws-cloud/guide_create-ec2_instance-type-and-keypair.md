# Guideline - Create an EC2 instance
## Guidelines: Choose Instance Type
1. Go to **EC2 Dashboard**.
    In the AWS Console, search for **EC2** and open the **EC2 Dashboard**.
2. Select **Instances**.
    In the left panel, click **Instances â†’ Launch instances**.
3. Choose an Amazon Machine Image (AMI):
    - Select an appropriate AMI (e.g., Amazon Linux 2 AMI).
4. Choose Instance Type:
    - Select an instance type (e.g., `t2.micro` for free tier).
5. Click **Next: Configure Instance Details**.
6. Configure Instance Details as needed.
7. Click **Next: Add Storage**.
8. Add Storage as needed.
9. Click **Next: Add Tags**.
10. Add Tags as needed.
11. Click **Next: Configure Security Group**.
12. Configure Security Group as needed.
13. Click **Review and Launch**.
14. Review your instance configuration.
15. Click **Launch**.
